# 동기
최근 1인가구와 배달의 수요가 폭발적으로 증가함으로 인하여 최소배달비가 올라갔습니다. 배달비를 줄이는 것은 어렵기 때문에 배달료의 부담을 주변사람들과 함께 '배달 공동구매'를 하여 음식을 공유하면 좋겠다 라는 생각으로 만들었습니다.
